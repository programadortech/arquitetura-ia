# Assets do guia

Coloque aqui o **logo oficial da 2A** com o nome exato:

```
docs/guia/assets/logo-2a.png
```

O guia (`docs/guia/index.html`) referencia esse arquivo no header, no hero e no rodapé.
É o arquivo enviado pelo time — não recriado. Formato recomendado: PNG com fundo transparente
(ou o PNG original). Se preferir SVG, salve como `logo-2a.png` mesmo assim ou ajuste o `src` no HTML.
