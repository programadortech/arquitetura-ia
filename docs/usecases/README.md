# Casos de Uso

Catálogo de casos de uso da aplicação. Cada caso de uso é implementado com `/create-usecase` a partir de
[`templates/usecase-template.md`](../../templates/usecase-template.md) e segue
[`../standards/usecase-dispatcher.md`](../standards/usecase-dispatcher.md).

Cada entrada: nome, feature, request → response, ports utilizados, efeitos colaterais (DB/fila/job) e os
critérios de aceitação que satisfaz.
